# Contributing

Thanks for your interest in contributing to Neural Observatory!

## How to Contribute

1. **Fork** the repository and clone your fork locally.
2. Create a new branch for your feature or bugfix:

   ```bash
   git checkout -b feature/my-new-feature
   ```

3. Make your changes. Please keep the code clean and add comments where necessary.
4. Add tests for any new functionality in the `tests/` directory.
5. Ensure all tests pass and code is formatted:

   ```bash
   pytest tests/
   black .
   ```

6. Commit your changes and push to your fork.
7. Open a **Pull Request** on GitHub with a clear description of your changes.

## Reporting Bugs

If you find a bug, please open an issue on GitHub and include:

- A clear description of the problem.
- Steps to reproduce it.
- The error traceback (if any).

We appreciate your help!
